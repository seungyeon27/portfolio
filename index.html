Web Publisher

안녕하세요! 사람들에게 쉽게 원하는 정보를 드리는 웹 퍼블리셔 강승연입니다.

  • 광활한 인터넷 세상 속에서 많은 사람들이 쉽고 빠르게 원하는 정보를 찾는 것이 중요하다고 생각합니다. 저는 빠르고 정확하게 내용을 전달하는 웹을 만드는 것이 목표입니다.
  • 브랜드의 색에 맞는 디자인으로 브랜드의 가치를 높이고, 빠르고 정확하게 사용자의 경험을 제공하겠습니다.
  • 프로젝트 일을 하며 사람들과의 협업과 소통이 중요하다고 생각합니다. 원활한 소통을 할 수 있도록 융통성 있는 판단과 좋은 마무리를 할 수 있도록 노력하겠습니다.

Tech Stack

  • Photoshop을 이용한 배너, 상세페이지, UI 디자인 및 편집 작업이 가능합니다.
  • Illustrator를 이용한 로고, 아이콘 제작이 가능합니다.
  • Adobe XD를 이용한 와이어프레임, 스토리보드, UI/UX 설계와 프로토타이핑이 가능합니다.
  • 웹 표준(마크업,유효성), 웹 접근성, 크로스브라우징이 가능합니다.
  • 반응형, 모바일 웹 제작이 가능합니다.
  • HTML, CSS를 이용한 퍼블리싱이 가능합니다.
  • JAVASCRIPT, jQuery를 이용한 동적인 웹 구현이 가능합니다.
  • FTP filezila, github를 이용한 서버 관리, 프로젝트 관리가 가능합니다.
  • 사이트 조사/분석, 벤치마크, 정보구조설계(IA), 시나리오를 이용해 웹기획이 가능합니다.
  • Visual Studio Code, Edit plus를 활용할 수 있습니다.
